유형 체크지

각 문항을 1~5점으로 눌러 체크하면 합계와 상위 유형이 자동으로 계산돼요.

점수 기준

1점 전혀 그렇지 않다 · 2점 대체로 그렇지 않다

3점 보통이다 · 4점 대체로 그렇다 · 5점 매우 그렇다

0 / 80 문항 체크됨 0%
저장 자동 유지 상위 유형 자동 계산 결과 복사 가능

결과 요약

현재 체크한 점수를 기준으로 높은 순서를 보여줘요. 동점이면 같은 점수로 함께 표시돼요.

같은 휴대폰에서 다시 열어도 체크 내용이 남도록 자동 저장돼요.
